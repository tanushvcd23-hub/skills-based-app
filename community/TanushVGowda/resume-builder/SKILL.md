# Resume Builder Skill

## Purpose
Use this skill to create professional resumes.

## When to Use
- Fresher resume
- Internship resume
- ATS resume
- Resume improvement

## Inputs Required
- Full Name
- Phone
- Email
- Skills
- Education
- Projects
- Experience

## Instructions
1. Ask user for missing details.
2. Use professional formatting.
3. Keep concise bullet points.
4. Highlight achievements.
5. Make ATS friendly.

## Output Format
- Contact
- Summary
- Skills
- Education
- Projects
- Experience
